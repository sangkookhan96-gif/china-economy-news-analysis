# 뉴스 제목 번역 품질 개선 계획

**작성일**: 2026-02-07
**현황**: Google Translate(무료) 직역 → 사용자 수용성 저하

---

## 1. 문제 분석

### 1.1 현재 시스템
```
원문(중국어) → Google Translate API(무료) → 한국어 제목
```
- 비용: **무료** (deep_translator 라이브러리)
- 품질: **직역 수준**, 문맥/맥락 무시

### 1.2 발견된 문제 유형

| 유형 | 예시 (원문 → 현재 번역) | 문제점 |
|------|------------------------|--------|
| **인명 미번역** | 李强 → "Li Qiang" | 한글 음역 필요 (리창) |
| **직역 어색** | 预制菜国标 → "준비된 요리에 대한 국가 표준" | "예제요리 국가표준"이 자연스러움 |
| **중국 용어** | 两会 → "두 번의 세션" | "양회"로 의역 필요 |
| **관점 표현** | 我国 → "우리나라" | "중국"으로 치환 필요 (✅ 해결됨) |
| **장황한 문장** | 길고 복잡한 번역 | 18자 헤드라인에 부적합 |

### 1.3 근본 원인

1. **Google Translate 한계**: 뉴스 제목 특화 학습 없음
2. **도메인 지식 부재**: 중국 경제/정치 전문 용어 처리 불가
3. **후처리 부재**: 번역 후 검수/보정 단계 없음

---

## 2. 해결 방안 (비용순)

### 방안 A: 규칙 기반 후처리 (비용: 무료)

**원리**: 번역 후 정규식/사전으로 교정

```
Google 번역 → 규칙 기반 후처리 → 최종 제목
```

**구현 항목**:
| 항목 | 구현 방법 | 예상 효과 |
|------|----------|----------|
| 인명 사전 | 중국 주요 인물 한글 음역 매핑 | 높음 |
| 전문 용어 | 两会→양회, 国务院→국무원 등 | 높음 |
| 관점 치환 | 我国→중국, 国内→중국 (✅ 완료) | 높음 |
| 축약 규칙 | 불필요한 조사/수식어 제거 | 중간 |

**비용**: 0원
**효과**: 70% 문제 해결 예상

---

### 방안 B: 하이브리드 (무료 번역 + 선택적 AI 보정)

**원리**: 문제 감지 시에만 Claude API 호출

```
Google 번역 → 품질 검사 → [문제 시] Claude 보정
                        → [정상 시] 그대로 사용
```

**품질 검사 기준**:
- 영문 인명 포함 여부
- 제목 길이 > 30자
- 금지 패턴 감지 (준비된, ~에 대한 등)

**예상 API 호출률**: 전체의 20~30%
**비용**: 월 $1~3 수준 (뉴스 100건/일 기준)

---

### 방안 C: Claude 전용 (고품질, 고비용)

**원리**: 모든 제목을 Claude로 번역

```
원문 → Claude API (번역+의역+축약) → 최종 제목
```

**프롬프트 전략**:
- 한국인 독자 관점 번역
- 18자 이내 축약
- 중국 고유명사 한글 표기

**비용**: 월 $10~20 (뉴스 100건/일)
**효과**: 95% 품질 개선

---

## 3. 권장안: 방안 A + B 조합

### 3.1 구현 우선순위

| 단계 | 작업 | 비용 | 기대 효과 |
|------|------|------|----------|
| 1단계 | 인명/용어 사전 구축 | 무료 | 50% 개선 |
| 2단계 | 규칙 기반 후처리 모듈 | 무료 | 20% 추가 개선 |
| 3단계 | 품질 검사 + 선택적 AI 보정 | 월 $1~3 | 20% 추가 개선 |

### 3.2 예상 비용
- **즉시 적용 가능**: 무료 (방안 A)
- **최적 품질**: 월 $1~3 (방안 A+B)

### 3.3 구현 파일 구조

```
src/utils/
├── translator.py              # 기존 Google 번역
├── title_validator.py         # 관점 표현 교정 (✅ 완료)
├── title_postprocessor.py     # [신규] 규칙 기반 후처리
├── chinese_dictionary.py      # [신규] 인명/용어 사전
└── title_refiner.py           # [신규] AI 보정 (선택적)
```
