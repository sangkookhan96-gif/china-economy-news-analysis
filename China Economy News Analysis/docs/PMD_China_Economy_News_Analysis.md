## 1. 프로젝트 개요

### 1.1 프로젝트 명
**중국 경제 뉴스 분석 플랫폼 (China Economy News Analysis Platform)**

### 1.2 프로젝트 요약
AI 기반 자동 뉴스 수집과 전문가 분석을 결합한 중국 경제 뉴스 분석 플랫폼. 중국 대륙 내부 매체를 중심으로 첨단 기술 산업 뉴스를 실시간 수집하고, Claude AI의 분석과 전문가(한상국)의 코멘터리를 통합하여 제공한다.

### 1.3 핵심 정보

| 항목 | 내용 |
|------|------|
| **개발자** | 한상국 |
| **전문가** | 한상국 (개발자 겸임) |
| **기술 스택** | Python, Claude API, MCP Servers, Streamlit |
| **주요 특징** | AI 분석 + 전문가 코멘터리 통합 |
| **목표 완료일** | 9주 후 (약 2개월) |
| **MVP 완료일** | 3주 후 |

### 1.4 프로젝트 비전
> "중국 대륙 내부 시각으로 바라보는, 가장 정확하고 깊이 있는 중국 경제 뉴스 분석 플랫폼"

## 2. 문제 정의

### 2.1 현재 상황 (As-Is)

#### 2.1.1 시장 환경
- 중국 경제는 눈부신 속도로 변화 중
- 중국 정부가 육성하는 첨단 과학기술과 산업이 예측하기 어렵게 발전
- 반도체, AI, 신에너지, 양자컴퓨터 등 전략 산업의 급격한 성장

#### 2.1.2 기존 도구의 한계

| 기존 도구 | 문제점 |
|----------|--------|
| **Reuters** | 서방 시각 중심, 중국 내부 맥락 부족 |
| **SCMP** | 홍콩 기반, 대륙 내부 뉴스 커버리지 제한 |
| **일반 크롤링 앱** | 뉴스 중요도 판단 불가, 단순 수집만 |
| **번역 서비스** | 경제/산업 전문 용어 오역 빈번 |

#### 2.1.3 핵심 페인포인트

1. **정보 소스의 한계**
   - 해외 매체 의존으로 중국 내부 시각 누락
   - 중국 정부 정책 의도 파악 어려움
   - 실제 산업 현장의 움직임 포착 불가

2. **분석 품질의 한계**
   - 진정한 중국 경제 전문가 부재
   - 피상적 번역 수준의 분석
   - 맥락 없는 단편적 뉴스 나열

3. **시의성의 한계**
   - 중요 뉴스 발생 후 인지까지 시간 지연
   - 수동 모니터링의 비효율성
   - 트렌드 변화 조기 포착 어려움

### 2.2 목표 상황 (To-Be)

#### 2.2.1 해결 목표

| 문제 | 해결 방안 |
|------|----------|
| 해외 매체 의존 | 중국 대륙 내부 매체 직접 수집 |
| 중요도 판단 불가 | AI 기반 자동 중요도 평가 |
| 전문가 분석 부재 | 한상국 전문가 코멘터리 통합 |
| 시의성 부족 | 실시간 자동 수집 (매 1시간) |

#### 2.2.2 핵심 원칙
- **중국 내부 매체 중심**: 해외 매체는 중요도 판단의 참고 자료로만 활용
- **AI + 전문가 협업**: AI의 효율성과 전문가의 통찰력 결합
- **첨단 기술 분야 집중**: 산업 범위를 명확히 한정하여 깊이 있는 분석

## 3. 타겟 사용자

### 3.1 주요 타겟 (Primary Users)

#### 3.1.1 기관 투자자
- **프로필**: 자산운용사, 증권사, 헤지펀드의 중국 담당 애널리스트
- **니즈**: 투자 판단을 위한 정확한 트렌드 정보
- **사용 빈도**: 일 1-2회 대시보드 확인
- **핵심 기대**: 시장에 영향을 줄 정책/기업 뉴스 조기 파악

#### 3.1.2 기업 전략팀
- **프로필**: 중국 사업 담당, 해외사업본부, 전략기획실
- **니즈**: 산업 동향 및 경쟁 환경 분석
- **사용 빈도**: 주 3-5회 심층 분석 활용
- **핵심 기대**: 중국 시장 진출/협력 전략 수립 근거

### 3.2 타겟 제외 (Non-Target Users)

| 제외 대상 | 제외 이유 |
|----------|----------|
| 언론인 | 속보 중심, 분석보다 팩트 필요 |
| 개인 투자자 | 전문 용어/분석 수준 불일치 |
| 일반 대중 | 중국 경제 관심도 낮음 |
| 학생/연구자 | 학술적 관점과 실무 관점 차이 |

### 3.3 사용자 요구사항

#### 3.3.1 언어 요구사항
- **입력**: 중국어 원문 (크롤링)
- **출력**: 한국어 번역 필수
- **품질 기준**: 경제/산업 전문 용어 정확한 번역

#### 3.3.2 최종 산출물 우선순위

```
1순위: 트렌드 파악
        └─ 중국 첨단 산업의 방향성 이해
        └─ 정책 변화의 의미 해석
        └─ 산업별 성장/위축 신호 포착

2순위: 투자 판단 근거
        └─ 특정 기업/산업 투자 의사결정 지원
        └─ 리스크 요인 조기 발견
        └─ 시장 영향 예측
```

### 3.4 사용자 페르소나

#### 페르소나 1: 김철수 (기관 투자자)
```
직책: A자산운용 중국펀드 매니저
경력: 금융권 12년, 중국 담당 5년
중국어: 불가능 (번역 의존)
니즈: 매일 아침 중국 시장 브리핑 필요
페인: "중요한 정책 발표를 뒤늦게 알게 됨"
기대: "AI가 중요 뉴스를 선별해주고, 전문가가 의미를 해석해주면 좋겠음"
```

#### 페르소나 2: 박영희 (기업 전략팀)
```
직책: B전자 해외전략팀 과장
경력: 전략기획 8년, 중국 시장 조사 3년
중국어: 기초 회화 가능, 뉴스 읽기 불가
니즈: 월간 중국 반도체 산업 동향 리포트 작성
페인: "신뢰할 수 있는 중국 내부 정보 소스가 없음"
기대: "중국 정부 정책 의도를 이해하고 싶음"
```

## 4. 핵심 가치 제안

### 4.1 Value Proposition Canvas

#### 4.1.1 고객의 과업 (Customer Jobs)
- 중국 경제 동향 파악
- 투자/사업 의사결정
- 정책 변화 모니터링
- 경쟁 환경 분석

#### 4.1.2 고객의 불편함 (Pains)
- 신뢰할 수 있는 정보 소스 부재
- 중국어 장벽
- 정보 과부하 (중요 뉴스 선별 어려움)
- 맥락 없는 단편적 뉴스

#### 4.1.3 고객의 기대 (Gains)
- 정확한 트렌드 파악
- 시간 절약
- 전문가 수준의 인사이트
- 경쟁 우위 확보

### 4.2 핵심 차별화 요소

| 차별화 요소 | 기존 도구 | 본 플랫폼 |
|------------|----------|----------|
| **정보 소스** | 해외 매체 중심 | 중국 대륙 내부 매체 |
| **분석 주체** | 단순 번역/요약 | AI + 전문가 협업 |
| **분야 집중** | 범용 경제 뉴스 | 첨단 기술 산업 특화 |
| **맥락 제공** | 단편적 뉴스 | 정책-기업-산업 연결 |
| **언어** | 영어 또는 중국어 | 한국어 완역 |

### 4.3 Unique Selling Point (USP)

> **"중국 대륙 내부 매체를 직접 분석하여, AI의 효율성과 전문가의 통찰력을 결합한 유일한 한국어 중국 경제 뉴스 분석 플랫폼"**

**FR-1.5: 중요도 평가**

| 항목 | 상세 |
|------|------|
| **설명** | AI가 뉴스의 중요도를 0.0-1.0 점수로 평가한다 |
| **우선순위** | 필수 (Must Have) |
| **평가 기준** | 정책 영향력, 산업 파급력, 시장 영향 |

**중요도 등급:**

| 점수 범위 | 등급 | 설명 |
|----------|------|------|
| 0.8-1.0 | 매우 중요 | 즉시 확인 필요, 시장 영향 큼 |
| 0.6-0.8 | 중요 | 당일 확인 권장 |
| 0.4-0.6 | 보통 | 참고용 |
| 0.0-0.4 | 낮음 | 아카이브용 |

**수락 기준:**
- [ ] 모든 뉴스에 중요도 점수 부여
- [ ] 중요도 분류 정확도 80% 이상 (인간 평가)
- [ ] 일일 Top 10 뉴스 자동 선정

**분류 체계:**

```
산업별 분류:

CREATE TABLE news (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    source VARCHAR(100) NOT NULL,           -- 출처 매체
    original_url TEXT NOT NULL,             -- 원문 URL
    original_title TEXT NOT NULL,           -- 원문 제목 (중국어)
    original_content TEXT,                  -- 원문 본문 (중국어)
    translated_title TEXT,                  -- 번역 제목 (한국어)

    related_news TEXT,                      -- 관련 뉴스 ID (JSON 배열)
    published_at DATETIME,                  -- 원문 발행일
    collected_at DATETIME DEFAULT CURRENT_TIMESTAMP,  -- 수집일
    analyzed_at DATETIME,                   -- 분석 완료일
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 인덱스
CREATE INDEX idx_news_importance ON news(importance_score DESC);
CREATE INDEX idx_news_industry ON news(industry_category);

# ai_analyzer.py 핵심 구조

class AIAnalyzer:
    def __init__(self):
        self.client = anthropic.Anthropic()
        self.model = "claude-sonnet-4-20250514"

    def analyze_news(self, news: dict) -> dict:
        """뉴스 분석 메인 함수"""
        return {
            "translation": self._translate(news),
            "summary": self._summarize(news),
            "importance": self._evaluate_importance(news),
            "category": self._classify(news),
            "sentiment": self._analyze_sentiment(news),
            "market_impact": self._predict_impact(news)
        }

    def generate_expert_comment(self, news: dict, analysis: dict) -> str:
        """전문가용 상세 AI 코멘트 생성"""
        prompt = f"""
        다음 중국 경제 뉴스에 대한 상세 분석 코멘트를 작성해주세요.

        [뉴스 정보]
        제목: {news['title']}
        내용: {news['content']}

        [기존 분석]
        {analysis}

        [코멘트 구조]
        1. 핵심 포인트 (3가지)
        2. 맥락 분석
        3. 시장 영향 예측
        4. 주목할 점
        """
        return self._call_api(prompt)

18개월   [Phase 6: 프리미엄 시작]
   ↓      기업 고객 확보
24개월   [Phase 7: 아시아 확장]
   ↓
3년+     [Phase 8: 글로벌 확장]
```

### 10.6 성장 전략 원칙

#### 10.6.1 집중과 선택

**반드시 할 것:**
- 중국 경제 뉴스 품질 최고 수준 유지
- 전문가 한상국의 인사이트 차별화
- 안정적인 자동화 시스템

**하지 않을 것:**
- 초기 무리한 확장
- 품질 저하시키는 빠른 성장
- 경쟁사 모방 (우리만의 강점 집중)

#### 10.6.2 성장 우선순위

```
1순위: 품질 (뉴스 분석의 정확성, 전문가 해석)
2순위: 안정성 (24시간 무중단 운영)
3순위: 사용자 경험 (직관적 UI, 빠른 로딩)
4순위: 확장 (기능 추가, 사용자 증가)
5순위: 수익화 (품질 확보 후)
```

#### 10.6.3 새 기능 추가 의사결정 기준

```
1. 이것이 핵심 가치를 높이는가? (중국 경제 인사이트)
2. 유지보수 부담이 적은가?
3. 한상국님 혼자 관리 가능한가?
4. 사용자에게 실질적 가치가 있는가?

→ 4개 질문에 모두 YES: 추가
→ 하나라도 NO: 보류
```
